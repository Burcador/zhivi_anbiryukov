# „Живи“, интерактивная повесть

Все персонажи этой истории соответствуют реально жившим людям.

Кроме главного героя. Его образ собирательный.

Включи звук. Здесь есть музыка.

В книге есть 10 концовок, 3 из них хорошие. Каждый твой выбор делает шаг к одной из этих концовок. С твоим продвижением по сюжету круг возможных концовок сужается, пока не останется только одна.

Автор текста - Александр Бирюков.
